# Papirus Playwright

---

## Demo in Playwright:

- For the first time use, remember:

```javascript
		npm i
```

- Create .env

```
	PAPIRUS_HOST=
	PAPIRUS_USER=
	PAPIRUS_PASSWORD=
```

- Run tests:

```javascript
		npm run test
```

Enjoy Testing! :)
